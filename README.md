# ML-Example

## Python 환경

- anaconda 4.6.11
  - python 3.6
  - numpy 1.17.3
  - matplotlib 3.1.1
  - tensorflow 2.0.0



## 환경 설정

```
>>> conda create -n python-env python=3.6
>>> conda activate python-env
>>> pip install numpy
>>> pip install matplotlib
>>> pip install tensorflow

혹은

>>> conda activate python-env
>>> pip install -r requirments.txt
```
